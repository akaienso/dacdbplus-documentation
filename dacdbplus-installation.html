Installation

Using WordPress:

  • Log in to your WordPress admin dashboard.
  • Go to "Plugins" and click on "Add New".
  • Click on "Upload Plugin" and choose the downloaded plugin file (.zip format) from your computer.
  • Click "Install Now" and wait for the installation to complete.

Manual Installation:

Alternatively, you can install the plugin via FTP:

  • Log in to your website's FTP account using an FTP client (e.g., FileZilla).
  • Access your website's root directory.
  • Navigate to the "wp-content/plugins" folder.
  • Extract the downloaded plugin file (.zip format) on your computer.
  • Upload the extracted plugin folder to the "wp-content/plugins" directory on your website using FTP.
  • Once uploaded, go to your WordPress admin dashboard.
  • Go to "Plugins" and locate the newly uploaded plugin.

Final Steps:

  1. Click "Activate" to activate the plugin.
  2. Update the plugin settings:
    • Click Settings under the plugin name on the Installed Plugins page.
    • In the Admin Dashboard menu, click Settings then DACdb.
  3. Enter your Account or District number.
  4. Enter your Club/Chapter/Region number. If you are managing a district-level website, set this to 0.
  5. Select your system:
    • Rotary: Choose DACdb (This is the default).
    • All others: Choose iMembersDB.
  6. Adjust other settings as needed to customize the display of data on your website.

IMPORTANT: If you intend to display maps on your website, you need to add a Google Maps API key to your settings.

That's it! DACdb Plus is now installed and active on your website!
